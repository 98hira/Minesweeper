- Python.3.7.3

#### 依存ライブラリ

- Pillow(PIL)
  →画像処理ライブラリ
  ```
  pip install pillow
  ```

- Tkinter
  → Tk(クロスプラットフォーム対応のGUI群)をpythonで使うためのライブラリ
